Для начала работы необходимо распаковать файлы в каталог поддержки 
AutoCAD. Обычно это что-то вроде: "%AutoCAD_HOME%\Support", где 
%AutoCAD_HOME% - каталог, в который установлен AutoCAD.

вызов: (tsp_draw)
